Morse Code Translator

Convert text to Morse code and back with ease. Perfect for learning or secret communication.

How to Use

Text to Morse

Simply type your message in plain text, and it will be converted to Morse code. Letters, numbers, and common punctuation are supported.

Morse to Text

Enter Morse code with spaces between letters and " / " between words. For example: ".... . .-.. .-.. --- / .-- --- .-. .-.. -.."

Switch Modes

Use the switch button to toggle between conversion modes. Your previous output becomes the new input automatically.

Morse Code Basics

Dots (.) are short signals, dashes (-) are long signals. Each letter has a unique pattern of dots and dashes.
