Morse Code Translator

A text-based Python program to convert Strings into Morse Code

Project Overview

This Morse Code Translator was built as part of my Python learning journey. It demonstrates string manipulation, dictionary usage, and basic Flask web development concepts for creating a simple but functional web application.

Features

  • Convert text to Morse code
  • Convert Morse code back to text
  • Clean, responsive web interface
  • Input validation and error handling
  • Support for letters, numbers, and common punctuation

Technologies Used

Python Flask HTML/CSS

Learning Outcomes

This project strengthened my understanding of string manipulation, dictionary data structures, and encoding/decoding logic. Using Flask, I learned how to handle routes, forms, and templates to build an interactive web app. It also introduced me to input validation, error handling, and the basics of the request-response cycle in web development.
